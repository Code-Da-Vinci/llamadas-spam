Sobre esta página

A tener en cuenta:

* Esta página está diseñado solo para exhibir esos números spam molestosos.

* Esta página no recoge los números mostrados para gestionar algún tipo de denuncia o sanción a las empresas y/o personas responsables de dichas llamadas.

* La idea principal de esta página es que si se recolecta datos suficientes, los organismos en contra de este tipo de prácticas puedan usarlos para hacer sus respectivas investigaciones y así sancionar a los responsables.

* Si lo que busca es denunciar al responsable de manera oficial, puede ingresar a este link: https://sgd.minjus.gob.pe/sgd-virtual/public/ciudadano/ciudadanoMain.xhtml

* Para hacer una publicación es necesario tener una cuenta de Facebook, para evitar publicaciones con malas intensiones.

* Se recomienda estar totalmente seguro que dichos números publicados sean realmente spam.
